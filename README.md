 📊 School Management Dashboard

 📌 Overview
This project is an interactive **Power BI dashboard** designed to analyze and visualize key metrics of a school system. It helps administrators monitor student performance, attendance, and overall school operations effectively.

 🎯 Objectives
- Track student performance
- Analyze attendance trends
- Monitor school-wise data
- Provide data-driven insights for decision making

 🚀 Features
- 📈 Student Performance Analysis
- 🏫 School-wise Insights
- 📅 Attendance Tracking Dashboard
- 📊 Interactive Filters & Visualizations
- 📌 KPI Indicators for quick insights

 🛠️ Tech Stack
- Power BI
- Data Cleaning & Transformation
- DAX (Data Analysis Expressions)

 📂 Dataset
The dataset includes:
- Student details
- Attendance records
- School registry data
- Teacher details
- Assessment data

 ▶️ How to Use
1. Download the `.pbix` file
2. Open it using **Power BI Desktop**
3. Explore dashboards using filters and slicers
